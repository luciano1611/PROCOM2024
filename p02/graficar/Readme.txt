El trabajo se divide en dos partes:

1. La primera es la función que realiza el ploteo 'figPlot', 
la cual posee su propio código para ser probada por separado. 
Esta utiliza las funciones alojadas en la carpeta 'func_ejercicio_A'.

2. La segunda parte es la función que actúa como interfaz con 
el usuario 'graficar', la cual hace uso de las funciones alojadas 
en la carpeta 'func_ejercicio_B'.

Para ejecutar los códigos, se utilizan los siguientes comandos:

Para el primer punto:
    ipython3 figPlot.py 
    o
    python3 figPlot.py 

Para el segundo punto:
    ipython3 graficar.py 
    o
    python3 graficar.py
